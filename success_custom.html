Custom Domain Setup

Point your domain to your Ask page

Add a CNAME record in your DNS provider to map your domain (e.g. ask.yourdomain.com) to your Ask page. Then email us and we'll configure the routing on our end — usually done within a few hours.

1
Log in to your domain provider and go to your DNS settings.
2
Add a new CNAME record using the values below.
3
Email us at hello@menoslabs.com with your custom domain and your Ask slug — we'll activate it.
Type Name / Host Value / Target TTL
CNAME
ask
ask-domains.menoslabs.com
Auto

DNS changes can take up to 24 hours to propagate, though usually much faster.

My Products → your domain → DNS → Add New Record → Type: CNAME → Name: ask → Value: the target above → Save.

Domain List → Manage → Advanced DNS → Add New Record → Type: CNAME → Host: ask → Value: the target above → Save.

DNS → Add Record → Type: CNAME → Name: ask → Target: the value above → Proxy: OFF (DNS only) → Save.

Domains → your domain → DNS Settings → Add Record → Type: CNAME → Host: ask → Data: the target above → Save.

Find your DNS management panel, add a CNAME record with host ask pointing to the target value above. If you're unsure, email us and we'll walk you through it.

Once your CNAME is set, email hello@menoslabs.com with your domain and Ask slug. We'll configure routing and confirm when it's live — usually within a few hours.
